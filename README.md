A fast, responsive web application that converts speech into text in real-time using the **Web Speech API**. It features a modern "Glassmorphism" UI, multi-language support, and local storage to remember your preferences.

## Live Demo

**[Live Demo](https://naveenapriya-06.github.io/voice-to-text-converter/)**

## Features

* **Real-time Transcription:** Converts voice to text instantly as you speak.
* **Multi-Language Support:** accurate recognition for English (US), Hindi, Spanish, French, and Tamil.
* **Dark Mode :** Toggle between light and dark themes.
* **Download Support:** Export your transcribed text as a `.txt` file with one click.
* **Responsive Design:** Works seamlessly on desktops and mobile devices.
